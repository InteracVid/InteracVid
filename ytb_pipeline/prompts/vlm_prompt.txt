# VLM Analysis Prompt

You are analyzing a video segment to understand the visual context of a reaction.

## Context
The speaker said: "{TRANSCRIPT}"
Chat message that triggered this: "{BULLET}"

## Task
Analyze the video segment and provide:
1. Visual description of the scene
2. Speaker's appearance and actions
3. Emotional state and body language
4. Any relevant visual elements (text on screen, overlays, etc.)

## Output Format
Return a JSON object:
```json
{
  "visual_description": "Description of the visual scene",
  "speaker_appearance": "Description of speaker's appearance",
  "body_language": "Description of gestures and expressions",
  "on_screen_text": "Any visible text or overlays",
  "emotional_tone": "Detected emotional state"
}
```

## Output
Return only valid JSON, no additional text.
